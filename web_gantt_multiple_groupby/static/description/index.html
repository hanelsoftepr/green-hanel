Web Gantt - Multiple GroupBy

This module allow users as well as developers group tasks by multiple fields and add description on each task by fields.

Main Features:

Allow to group task by one or more fields:

Example for adding a description by fields: 
    <gantt string="Planning Panel" date_start="planned_date" date_stop="expected_date" progress="progress_rate">
        <field name="planned_quantity"/>
        <field name="product_uom"/>
    </gantt>
